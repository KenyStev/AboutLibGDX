LibGDX

LibGDX es una libreria de codigo abierto escrita en Java para el Desarrollo de Videojuegos multiplataforma.

Plataformas como:

  • Linux / Android
  • HTML5
  • Mac / IOS
  • Windows

Algunas Razones para Usar LibGDX

Su Portabilidad

Algo de Historia

libGDX nació al final del 2009 cuando obtuve mi primer dispositivo Android. Desarrollando para Android era un dolor en aquel entonces. Así que me ideé una manera para desarrollar más que todo en máquinas de escritorio para no tener que esperar mucho tiempo para probar cambios en el código. Eso eventualmente se convirtió en libGDX (Mario Zechner)
